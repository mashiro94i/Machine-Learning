## 本機預覽

直接以瀏覽器開啟 `index.html` 或 `基礎.html` 即可。

## GitHub Pages

啟用後網址通常為：

**https://mashiro94i.github.io/Machine-Learning/**

### 啟用步驟（若尚未開啟）

1. 打開 Repo → **Settings** → **Pages**
2. **Source**：Deploy from a branch
3. **Branch**：`main`，資料夾 **`/ (root)`**
4. 儲存後等待 1～3 分鐘
